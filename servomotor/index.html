Komponen

Rangkaian Motor DOL

Proteksi

MCB / MCCB

Pemutus sirkuit otomatis yang melindungi rangkaian dari arus lebih dan hubung singkat pada jalur utama 3 fase.

Switching

Kontaktor Magnet

Saklar elektromagnetik berdaya besar untuk menghubung/memutus beban motor secara otomatis melalui sinyal kontrol.

Proteksi

Thermal Overload Relay

Melindungi motor dari panas berlebih akibat beban lebih. Memutus rangkaian otomatis bila arus melebihi batas setpoint.

Kontrol

Push Button ON (NO)

Tombol hijau normally-open untuk menghidupkan motor. Mengaktifkan coil kontaktor sehingga kontak utama menutup.

Kontrol

Push Button OFF (NC)

Tombol merah normally-closed untuk mematikan motor. Memutus rangkaian kontrol sehingga kontaktor de-energized.

Tambahan

Auxiliary Contact

Kontak bantu kontaktor untuk self-holding circuit — mempertahankan motor tetap ON setelah tombol ON dilepas.

Beban

Motor Induksi 3 Fase

Beban utama yang berputar saat ketiga fase terhubung. Rating tipikal: 380 V, 50 Hz. Arus starting 5–7× arus nominal.

Materi

Sistem Kerja Motor 3 Fase

1
Motor Induksi 3 Fase
▶
Motor induksi 3 fase adalah mesin listrik yang mengubah energi listrik AC 3 fase menjadi energi mekanik rotasi. Prinsip kerjanya berdasarkan induksi elektromagnetik — medan magnet putar (rotating magnetic field) yang dihasilkan stator menginduksi arus pada rotor sehingga rotor ikut berputar.
  • Stator: bagian diam dengan belitan 3 fase
  • Rotor: bagian berputar (squirrel cage atau wound rotor)
  • Kecepatan sinkron: Ns = 120f/P (f=frekuensi, P=jumlah kutub)
2
Rangkaian Direct On Line (DOL)
▶
DOL adalah metode starting motor paling sederhana di mana motor langsung dihubungkan ke tegangan jaringan penuh tanpa rangkaian pereduksi tegangan.
  • Cocok untuk motor kecil hingga ±5,5 kW
  • Arus starting tinggi (5–7× I nominal)
  • Torsi starting besar dan langsung
  • Komponen: MCB + Kontaktor + TOR + Push Button ON/OFF
Keuntungan: murah, sederhana, torsi langsung besar.
Kekurangan: lonjakan arus besar saat starting.
3
Tegangan & Arus Fase
▶
Tegangan Fase (V Phase) adalah tegangan antara salah satu konduktor fase dan titik netral. Pada sistem 380/220 V Indonesia:
  • Tegangan fase-netral: 220 V nominal
  • Tegangan fase-fase: 380 V nominal
  • Toleransi: ±10% (198–242 V)
Arus Fase (I Phase) mengalir pada masing-masing konduktor fase. Monitoring arus penting untuk mendeteksi beban lebih, putusnya fase, dan kondisi abnormal lainnya.
4
Ketidakseimbangan Fase
▶
Ketidakseimbangan tegangan terjadi bila tegangan antar fase tidak sama. Standar NEMA MG-1 mensyaratkan ketidakseimbangan ≤ 1% untuk operasi normal.
  • Formula: % Imbalance = (Deviasi maks / Rata-rata) × 100%
  • Efek: panas berlebih pada belitan, getaran, umur motor berkurang
  • Setiap 1% ketidakseimbangan menaikkan suhu ±1–2%
Ketidakseimbangan arus lebih berbahaya dari ketidakseimbangan tegangan dan sering disebabkan oleh beban tidak merata.
5
Sistem Proteksi Motor
▶
Proteksi motor bertujuan melindungi motor dan rangkaian dari kondisi abnormal:
  • MCB/MCCB: proteksi hubung singkat & arus lebih pada rangkaian utama
  • Thermal Overload Relay: proteksi beban lebih berkelanjutan (panas)
  • Fuse: proteksi hubung singkat cepat
  • Thermistor/PTC: proteksi suhu belitan langsung
Kombinasi MCB + Kontaktor + TOR memberikan perlindungan menyeluruh untuk aplikasi DOL standar industri.
6
Monitoring IoT via Google Sheets
▶
Sistem ini menggunakan Google Sheets sebagai database IoT untuk menyimpan dan menampilkan data sensor secara real-time.
  • Hardware: ESP32/ESP8266 + sensor tegangan (ZMPT101B) + sensor arus (SCT-013)
  • Firmware: membaca ADC dan mengirim data ke Sheets via HTTP POST (Google Apps Script)
  • Sheets API v4: dashboard ini membaca baris terbaru sebagai nilai live
  • Format kolom: A=Timestamp, B=V1, C=I1, D=V2, E=I2, F=V3, G=I3
Auto-refresh tersedia setiap 5, 10, 30 detik atau 1 menit sesuai kebutuhan pemantauan.
